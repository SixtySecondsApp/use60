## Template 1: SaaS Hero Image (Landscape)

The bread and butter. Use for landing page hero sections.

```
A [SUBJECT — e.g. "modern team of 3 diverse professionals collaborating around a floating
holographic dashboard interface"] in a [STYLE — e.g. "cinematic, photorealistic"] style.
[ACTION — e.g. "The lead figure gestures toward a data visualization, others lean in with engaged expressions"].
[COMPOSITION — e.g. "Wide angle shot, slightly low perspective, three-quarter view"].
[ENVIRONMENT — e.g. "In a sleek, glass-walled corner office with panoramic city views at golden hour"].
[LIGHTING — e.g. "Warm golden hour light from floor-to-ceiling windows on the right,
complemented by cool blue glow from the holographic interface, creating dramatic color contrast"].
[TECHNICAL — e.g. "Shot on 35mm lens, f/2.8, shallow depth of field with bokeh on city lights,
slight warm color grade, highly detailed textures, 4K"].
Color palette: [HEX CODES with usage — e.g. "#8129D7 violet accents on the interface,
#2A5EDB blue data visualizations, #F8FAFC light backgrounds"].
Clean, high-end SaaS aesthetic. No text, no watermarks, no logos.
```

## Iteration Checklist

When a result isn't right, change ONE layer at a time:

| Problem | Layer to Change | Example Adjustment |
|---------|----------------|-------------------|
| Wrong mood/feeling | Layer 5: Lighting | "golden hour" → "cool blue studio lighting" |
| Wrong angle/framing | Layer 3: Composition | "eye level" → "dramatic low angle" |
| Wrong art style | Layer 6: Style | "photorealistic" → "flat vector illustration" |
| Too busy/cluttered | Layer 4: Environment | Simplify description, add "minimal, clean" |
| Low quality/blurry | Layer 7: Technical | Add "highly detailed, 4K, masterpiece quality" |
| Wrong colors | Add hex codes | Specify exact palette with usage instructions |
| Subject unclear | Layer 1: Subject | Add more physical details, clothing, expression |
